Design Philosophy

  • Safety-first content: Educational, no medical claims; disclaimers on every page.
  • Plain language: Short, scannable sections; consistent headings and icons.
  • Accessibility: Skip links, labeled controls, sufficient color contrast, keyboard-friendly UI.
  • Performance: Minimal JS, CDN Bootstrap, deferred scripts, optimized assets.
  • Soft personalization: LocalStorage (no login) to reorder tips/products by dosha.
  • Consistent theming: Shared `theme.css` variables for colors/spacing/rounded corners.

Tech stack: HTML, CSS (Bootstrap + custom), light JS. No backend required.
